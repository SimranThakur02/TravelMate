# TravelMate 🌍✈️

TravelMate is a modern travel companion platform designed to help travelers explore destinations, optimize packing, manage expenses, and plan smarter trips from a single application.

## Features

### 🌎 Destination Explorer

* Search any destination worldwide
* View destination information
* Learn history and cultural significance
* Discover popular attractions
* Check weather conditions
* Explore destination images

### 🎒 PackLight

* Smart packing assistant
* Luggage weight management
* Weather-based packing suggestions
* Airline baggage limit tracking

### 💰 RoamBudget

* Travel expense tracking
* Multi-currency support
* Split expenses with friends
* Spending analytics and insights

### ✈️ Layover Roulette

* Smart layover planner
* Calculate available free time
* Discover nearby attractions
* Optimize short stopovers

### 🏙️ NomadScout (Upcoming)

* Digital nomad neighborhood finder
* Work-friendly café recommendations
* Community reviews
* Remote work resources

## Installation

Clone the repository:

```bash
git clone https://github.com/SimranThakur02/TravelMate.git
```

Navigate to the project folder:

```bash
cd travelmate
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```
